npm install
¿Qué hace? Instala las nuevas dependencias de Capacitor que acabamos de agregar a tu proyecto.
npm run build
¿Qué hace? Compila tu aplicación web usando Vite y crea la carpeta dist. Capacitor necesita esta carpeta para funcionar.
npx cap add android
¿Qué hace? Crea la carpeta android en la raíz de tu proyecto. Esta carpeta contiene un proyecto completo de Android nativo.
npx cap sync
¿Qué hace? Copia tu aplicación web compilada (de la carpeta dist) dentro del proyecto de Android. Debes ejecutar este comando cada vez que hagas cambios en tu aplicación web.
npx cap open android
¿Qué hace? ¡Este es el paso mágico! Abrirá automáticamente Android Studio con tu nuevo proyecto nativo cargado.



npm install @capacitor/cli --save-dev

npm install @capacitor/core

npm install @capacitor/android @capacitor/ios

npx cap init reportes-app com.macris.reportes --web-dir dist

npx cap init: Inicia la configuración de Capacitor.
reportes-app: Es el nombre de tu aplicación.
com.macris.reportes: Es un identificador único para tu app, crucial para las tiendas de aplicaciones.
--web-dir dist: Le indica a Capacitor que los archivos de tu aplicación web (la versión final y optimizada) se encontrarán en una carpeta llamada dist.
Después de ejecutarlo, deberías ver un nuevo archivo en tu proyecto llamado capacitor.config.ts.
