Instrucciones de Importación

Para cargar múltiples equipos exitosamente, su archivo Excel (.xlsx) o CSV debe incluir una tabla (en la primera hoja) con encabezados en la primera fila. Se recomienda seguir esta estructura de columnas:

  1. Placa u ID Manual
  2. Marca (Obligatorio)
  3. Modelo (Obligatorio)
  4. Capacidad (Ej: 12000 BTU)
  5. Ciudad (Obligatorio)
  6. Empresa (Obligatorio)
  7. Sede (Opcional, pero debe pertenecer a la Empresa)
  8. Dependencia (Opcional, pero debe pertenecer a la Empresa/Sede)
  9. Tipo de Equipo (Ej: Mini Split, Central, etc.)
  10. Refrigerante (Ej: R410A)
El sistema leerá las columnas dinámicamente si los nombres (encabezados) coinciden. Los datos de ciudades, empresas y tipos deben estar literalmente igual a como están registrados en el sistema para evitar ser marcados con error.
